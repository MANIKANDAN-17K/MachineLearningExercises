# 🤖 Machine Learning Exercise

Welcome to **MachineLearningExercise** 👨‍💻  

This repository contains practical Machine Learning exercises and mini-projects built while learning ML from **fundamentals to advanced concepts**.

The main goal of this repository is to strengthen theoretical understanding through implementation.

---

## 📚 Topics Covered

This repository includes exercises on:

### 🟢 Fundamentals
- Data Preprocessing
- Data Cleaning
- Feature Engineering
- Train-Test Split
- Model Evaluation Metrics

### 🟡 Supervised Learning
- Linear Regression
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Naive Bayes
- Support Vector Machine (SVM)
- Decision Tree
- Random Forest

### 🔵 Unsupervised Learning
- K-Means Clustering
- Hierarchical Clustering
- DBSCAN
- Apriori Algorithm (Association Rule Mining)

### 🔴 Advanced Topics
- Cross Validation
- Hyperparameter Tuning
- Feature Selection
- Model Comparison
- Performance Visualization

## 🚀 How to Run

1. Clone the repository:
```
git clone https://github.com/your-username/MachineLearningExercise.git
```

2. Install required libraries:
```
pip install -r requirements.txt
```

3. Run any Python file:
```
python filename.py
```
